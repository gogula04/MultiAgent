Alphanumeric: Set of characters or numbers

Fixed_Chars: Specific characters or choices from a fixed set

Each format variation should be listed as a separate object in the string_format array

Component constraints should be included in the constraints field

Number ranges should be specified with a hyphen (e.g., 0-360)

Alphanumeric ranges should be specified in brackets with a hyphen (e.g., [A-Z]; [0-9])
